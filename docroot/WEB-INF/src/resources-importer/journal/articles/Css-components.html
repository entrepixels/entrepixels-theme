Información importante

Este tema de apariencia de Liferay utiliza Bootstrap y AlloyUi, el funcionamiento correcto del tema de apariencia dependerá en parte de estas tecnologías.

Para más información sobre la base de css sobre el que se basa este tema de apariencia visitar las páginas web de Liferay Base-css y Componentes. También podéis visitar las páginas web de Bootstrap: Css y Componentes

Otros enlaces interesantes: Bootstrap Javascript y AlloyUi ejemplos y "Developer's Guide"

Algunas de las propiedades css no funcionarán correctamente en los navegadores antiguos como el ie7-8.
